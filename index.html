Plataforma de Universidad/Colegio

Deseo llevar a cabo una plataforma escolar donde se pueda tener el seguimiento de cada alumno en particular y ademas que desde alli se puedar cargar las notas. Tendria varias funcionalidades tanto para el alumno como para los profesores y tutores o preceptores. Se podran cargar notas, subir apuntes de la clase, programar clases virtuales, consultas al profesor por la plataforma, etc.

Este proyecto tambien puede aplicarse en una universidad realizando las correspondientes adaptaciones para que funcione correctamente. Hay algunas funcionalidades que todavia no puedo llevar a cabo pero seguro con el pasar de las clases las podre realizar.
